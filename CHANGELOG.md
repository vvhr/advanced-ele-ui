## [0.0.3] - 2025-11-06

### ✨ 新增

#### 核心组件
- **ZwForm** - 高级表单组件
  - 支持多种表单控件（Input、Select、DatePicker 等）
  - 内置自动校验规则（必填、邮箱、手机号等）
  - 支持动态表单项（显示/隐藏、禁用/启用）
  - 支持表格嵌套（可编辑表格）
  - 灵活的布局配置（栅格布局）
  - 丰富的插槽支持（前置、后置、自定义渲染）

- **ZwTable** - 高级表格组件
  - 多种列类型（索引、选择、操作、字典、日期、金额、敏感信息）
  - 支持行内编辑模式
  - 内置分页功能
  - 支持排序、筛选
  - 支持列固定
  - 金额格式化（千分位、小数位）
  - 敏感信息脱敏（手机号、身份证、邮箱）
  - 灵活的操作列配置

- **Icon** - 图标组件
  - 基于 Iconify，支持 10 万+ 图标库
  - 灵活的尺寸设置
  - 自定义颜色
  - 按需加载，性能优秀

#### 构建配置
- 配置 Vite 构建系统
- 支持 TypeScript 5.3
- 自动生成类型声明文件
- 支持 ES Module 和 UMD 格式输出
- 配置 Tree Shaking 优化

#### 开发工具
- 集成 ESLint + Prettier 代码规范
- 配置 UnoCSS 原子化 CSS
- 添加示例演示页面
- 完善的 TypeScript 类型定义

## 版本说明

### 版本号规则

遵循语义化版本规范 (Semantic Versioning):

- **主版本号 (Major)**: 不兼容的 API 修改
- **次版本号 (Minor)**: 向下兼容的功能性新增
- **修订号 (Patch)**: 向下兼容的问题修正

### 更新类型

- ✨ **新增** - 新功能
- 🐛 **修复** - Bug 修复
- 💄 **样式** - 样式调整
- ♻️ **重构** - 代码重构
- ⚡ **性能** - 性能优化
- 📝 **文档** - 文档更新
- 🔧 **配置** - 配置变更
- ⬆️ **依赖** - 依赖更新
- 🗑️ **废弃** - 废弃功能
- 💥 **破坏** - 破坏性变更
