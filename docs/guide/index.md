# 介绍

Advanced Ele UI 是一个基于 [Vue 3](https://v3.vuejs.org/) 和 [Element Plus](https://element-plus.org/) 的高级组件库。

它旨在解决中后台管理系统中常见的 CRUD 场景，通过封装 `AeForm` 和 `AeTable` 等核心组件，极大地减少了重复代码的编写。

## 特性

- 🚀 **高效**：通过配置化方式快速构建表单和表格。
- 📏 **规范**：统一的 UI 交互和代码风格。
- 🔧 **灵活**：支持插槽和自定义渲染，满足复杂需求。
- 📝 **类型安全**：完整的 TypeScript 支持。
